What Powers 8tracks?

by Sarah Edwards

8tracks is a music radio service that has placed an emphasis on user-created playlists, rather than those generated by an algorithm. 1 This has given a community of music lovers a platform to share playlists with one another and discover new artists.

So, what powers 8tracks?

Peter Bui, an infrastructure engineer at 8tracks, says that their stack is built on Amazon Web Services and that they use MySQL as their database. "We manage all our servers with Chef," he explains. The web servers themselves operate via nginx. They also utilize a number of Redis servers, though Bui doesn't go into detail. 2 8tracks is primarily programmed using Ruby on Rails, but Sinatra is used out of convenience in instances where less complexity is necessary. 3 When using 8tracks, users can search for up to three tags at once, in order to tailor their playlists to their current mood. For example, a user might search for, "indie rock," "happy," and "summer," to receive a list of results that fit each of the criteria. Any searches done by the user are handled by Solr. 2

8tracks uses RequireJS to improve load time for users, and BackboneJS as the basic framework for their web app. 3 As 8tracks does allow users to stream audio, it's unsurprising that all of their audio files are encoded in the stream-friendly AAC format. 4 The playback of all audio files is handled by Soundmanager 2, web software that supports "a truly ridiculous range of browsing platforms," effectively eliminating any incompatibility issues across browsers. 3

A partnership with Soundcloud allows users to search the library of tracks available through the Soundcloud service, rather than spending the time to upload their own tracks in order to create a playlist. Another partnership with imgix gives 8tracks a place to host images uploaded for playlist covers, and allows those images to be responsive across different devices. 3

It seems 8tracks is becoming increasingly committed to being "the source" for music discovery. In an effort to help users learn more about the artists they are listening to, 8tracks has decided to use an API called OpenAura that is filled with various artist biographies. 5 These biographies appear whenever a user is listening to a playlist. OpenAura actually allows artists to claim their "aura" and modify any of the information in order to promote accuracy. 6

8tracks has used all of these things to create a web app that is undoubtedly one of the best available for music lovers. It will be interesting to see how 8tracks continues to adapt future technology and see if its user base continues to expand.
